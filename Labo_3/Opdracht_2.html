Opdracht 2: DOM-tree Bewerken

  1. Als je een element via de Chrome Developer Tools uit de DOM-tree verwijdert, verdwijnt dat onderdeel direct uit de webpagina[web:2][web:8].
  2. Wanneer je tekst aanpast van een element in de DOM-tree, verandert de getoonde inhoud op de pagina meteen — zonder dat de broncode op de server wordt aangepast[web:2][web:5][web:8].
  3. Dit gebeurt volledig lokaal in de browser; er is geen interactie met de server nodig[web:2][web:5].
  4. Statische HTML-documenten blijven altijd zoals ze bij het laden waren. Bij dynamische HTML-documenten kan JavaScript de DOM aanpassen zodat de pagina verandert zonder te herladen — daardoor kunnen de broncode en de huidige DOM verschillen[web:6][web:9][web:15].
