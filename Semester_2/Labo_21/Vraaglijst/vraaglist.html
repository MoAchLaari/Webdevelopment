Opdracht - demo storage 1 - globale objecten

Blijft de data behouden als je de pagina herlaadt? Nee
Globale variabelen bestaan alleen zolang de pagina geladen is. Bij refresh wordt alles opnieuw gestart.

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent? Nee
Als je het tabblad sluit, verdwijnt de JavaScript-data uit het geheugen.

Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data? Nee
Elk tabblad laadt zijn eigen versie van de pagina met eigen globale variabelen.

Opdracht - demo storage 2 - session storage

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent? Nee
Session storage is gekoppeld aan één tabblad. Als dat tabblad sluit, wordt de data verwijderd.

Blijft de data behouden als je de pagina herlaadt? Ja
Bij een refresh blijft session storage bestaan, zolang hetzelfde tabblad open blijft.

Gebruiken twee tabbladen die dezelfde pagina tonen, dezelfde data? Nee
Elk tabblad heeft zijn eigen session storage.

Denk je dat een andere pagina, andere file of url, aan de data geraakt als je ze in hetzelfde tabblad opent? Ja
Ja, als die pagina van dezelfde website/origin komt. Session storage wordt gedeeld binnen hetzelfde tabblad en dezelfde origin.

Opdracht - demo storage 3 - local storage

Blijft de data behouden als je het tabblad sluit en de pagina opnieuw opent? Ja
Local storage blijft opgeslagen in de browser, ook na het sluiten van een tabblad.

Blijft de data behouden als je de pagina herlaadt? Ja
Een refresh verwijdert local storage niet.

Blijft de data behouden als je de browser afsluit, opnieuw start en de pagina weer opent? Ja
Local storage blijft bewaard tot je het zelf verwijdert of de browserdata wist.

Blijft de data behouden als je de pagina in een andere browser opent? Nee
Elke browser heeft zijn eigen local storage. Chrome deelt dit bijvoorbeeld niet met Firefox.

Wordt de data gedeeld door verschillende tabbladen als je de pagina meermaals opent? Ja
Local storage wordt gedeeld tussen tabbladen van dezelfde website/origin.
